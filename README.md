# Lux Hotel

Sistema administrativo para gerenciamento de hotel desenvolvido com HTML, CSS e JavaScript.

## Funcionalidades

- Login de administrador
- Dashboard com métricas
- Gráfico de reservas
- Sidebar recolhível
- Gestão de hóspedes
- Gestão de quartos

## Demonstração

🔗 Acesse o projeto:
https://luxhotelbr.netlify.app

## Tecnologias

- HTML5
- CSS3
- JavaScript
- Chart.js
